# Customer complaint: agent context drift

Source: operator-forwarded email
From: Riley Chen, VP Support at Acme Desk
To: Maya at Northstar Tools
Subject: Agent keeps forgetting our rollout context

Riley says Acme Desk's support team tried Northstar's agent workflow for two weeks. The agent handled simple ticket triage, but it repeatedly forgot that Acme was piloting only the enterprise queue and not the SMB queue.

The support lead had to paste the same rollout notes into three separate conversations. Riley says this erased the time savings and made the team nervous about expanding the pilot.

Customer quote: "The agent is useful when it remembers the operating context. When it forgets, it creates cleanup work for my leads."

Potential follow-up: Riley is open to another pilot week if Northstar can show how durable rollout context is stored, reviewed, and updated.
