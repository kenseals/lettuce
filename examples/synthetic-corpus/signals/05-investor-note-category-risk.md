# Investor note: category risk

Source: operator-forwarded investor email
From: Lena Ortiz, Seed investor
To: Maya at Northstar Tools
Subject: Question on agent memory category

Lena likes the idea of durable agent context, but worries the category will be swallowed by the large model providers as they add persistent automations.

She asks what Northstar owns that OpenAI, Anthropic, or Google would not simply bundle into their agent products.

Her suggested wedge: "Pick one painful workflow where context changes need to be reviewed and audited. Do not pitch a generic memory layer."

Potential strategic implication: Northstar should describe itself as the reviewable operating-context protocol for agent work, not as generic agent memory.
